#  Blinkit Sales Analysis Dashboard (Power BI)

##  Project Overview

This project analyzes **Blinkit sales data** using **Microsoft Power BI** to understand the overall business performance across different outlet types, outlet sizes, outlet locations, item categories, and fat content.

The dashboard provides clear insights into key business metrics and helps stakeholders make **data-driven decisions**.

---

##  Business Problem

The stakeholders did not have a clear view of important business metrics such as:

* Total Sales
* Number of Items Sold
* Average Sales
* Average Rating

Because of this, it was difficult to analyze the performance across different **outlet sizes, outlet types, locations, and years**, making it challenging for stakeholders to make data-driven decisions.

---

##  Tools & Technologies Used

* **Microsoft Power BI**
* **Power Query (ETL)**
* **DAX (Data Analysis Expressions)**

---

##  Data Preparation

The dataset was cleaned and transformed using **Power Query** before building the dashboard.

Key steps included:

* Handling missing values
* Correcting data types
* Transforming columns for analysis
* Preparing the dataset for visualization

---

##  Key Metrics (DAX Measures)

The following key performance indicators were created using **DAX**:

* **Total Sales**
* **Total Items**
* **Average Sales**
* **Average Rating**

---

##  Dashboard Visualizations

The Power BI dashboard includes several interactive visuals:

* **Card Visuals** – Display key KPIs such as Total Sales, Total Items, Average Sales, and Average Rating.
* **Bar Charts** – Analyze sales performance by item type and fat content.
* **Donut Charts** – Show sales distribution by outlet size and fat content.
* **Matrix Visual** – Compare outlet type sales performance.
* **Funnel Chart** – Analyze outlet location performance.
* **Line Chart** – Visualize year-wise sales trends.

---

##  Key Insights

From the dashboard analysis, the following insights were identified:

* **Fruits and vegetables generate the highest sales.**
* **Low-fat products contribute significantly to overall sales.**
* **Medium and small outlet sizes generate higher revenue.**
* **Tier 3 locations contribute the highest sales.**
* Sales peaked in **2018**, indicating demand fluctuations across years.

---

##  Business Recommendations

Based on the insights, the following recommendations were suggested:

* Focus marketing efforts on **high-performing product categories**.
* Maintain **higher inventory during peak sales periods**.
* Run **promotional campaigns during low-sales periods** to increase demand.
